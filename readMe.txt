파일 중에 textFile 폴더가 포함되어 있는데, 이 파일은 데이터베이스에서 텍스트 파일로 데이터를 각 테이블에 저장하기 위한 폴더입니다.

1. textFile 폴더를 통째로 C 드라이브에 넣는다.
2. MySql에서 실행을 해볼 서버에 대한 옵션에 들어간뒤, Advenced - Others 위치에 "OPT_LOCAL_INFILE=1" 이라는 문장을 추가한다(큰따옴표는 제외).
3. sql문을 위에서부터 차례대로 실행한다.

이대로 하시면, 데이터가 잘 들어가는지 확인하실 수 있습니다.

'DB 테이블 캡처본' 폴더에서 확인하실 수 있습니다.
